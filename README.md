# Student Buddy Chrome Extension

A productivity extension for students.

## Features

- Add Tasks
- Delete Tasks
- Save Tasks Permanently
- Task Counter
- Dark Mode
- Pomodoro Timer
- CGPA Calculator

## Technologies Used

- HTML
- CSS
- JavaScript
- Chrome Extension Manifest V3
- Git
- GitHub

## How to Run

1. Download the project.
2. Open Chrome.
3. Go to chrome://extensions
4. Enable Developer Mode.
5. Click Load Unpacked.
6. Select the project folder.
